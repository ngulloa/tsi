Mapa conceptual

Relación general entre los cuatro temas conceptuales.

El mapa muestra una ruta de lectura: contexto social, infraestructura digital, intercambio económico y creación de soluciones.

Ética, datos y sociedad

Define responsabilidades, límites y efectos sociales del uso de información. Ordena casos con cinco dimensiones: información, propiedad, control, calidad del sistema y calidad de vida.

→

Redes, Internet y movilidad

Entrega la infraestructura para conectar personas, sistemas, servicios y datos mediante protocolos, redes locales, Internet y acceso móvil.

→

Comercio digital

Usa la conectividad para crear mercados, plataformas y productos digitales.

→

Creación y cambio de sistemas

Convierte necesidades organizacionales en soluciones que deben adoptarse y evaluarse.

La relación no es lineal en la práctica: cada nuevo sistema puede generar nuevos datos, riesgos éticos, necesidades de red y oportunidades de comercio digital.

Volver al inicio

Para redes, sigue una cadena simple: dispositivo, medio de conexión, protocolo, red local, Internet, servicio y proceso organizacional afectado. Eso ayuda a no confundir infraestructura con aplicación.

Repasar redes

Para estudiar casos, parte por ética: identifica datos usados, actores afectados, beneficios, daños y responsables. Luego conecta con infraestructura, comercio o creación de sistemas según corresponda.
