Glosario

Consulta rápida de conceptos de TSI.

Úsalo como consulta rápida, no como ruta principal de estudio. Las definiciones son breves para volver al concepto sin reemplazar las páginas temáticas.

A

Abuso computacional

Uso dañino, invasivo o irresponsable de sistemas de información, aunque no siempre sea delito.

Afiliación

Modelo de ingresos donde un sitio o plataforma recibe dinero por derivar tráfico, registros o ventas hacia otro proveedor.

Ajuste dinámico de precios

Cambio frecuente de precios según demanda, inventario, competencia, perfil del usuario o contexto de compra.

Administración de procesos de negocios

Gestión continua de procesos para analizarlos, rediseñarlos, ejecutarlos, medirlos y ajustarlos según objetivos organizacionales.

Alcance global

Capacidad de un canal digital para llegar a clientes, proveedores o audiencias fuera de un mercado local.

Ancho de banda

Capacidad de transmisión de datos de un enlace en un periodo determinado.

Análisis de sistemas

Estudio del problema, sistema actual, actores, alternativas y requerimientos que debe cumplir una solución de información.

Aplicación nativa

Aplicación creada para una plataforma móvil específica, con acceso directo a recursos del dispositivo como cámara, ubicación o notificaciones.

Aplicación Web móvil

Aplicación accesible desde navegador y adaptada a pantallas, interacción y condiciones de conectividad móvil.

Asimetría de información

Situación donde una parte de una transacción sabe más que otra sobre precio, calidad, riesgo o condiciones.

Automatización

Uso de tecnología para ejecutar con más eficiencia una tarea o procedimiento existente.

Auditoría posimplementación

Revisión posterior a la puesta en marcha para evaluar si el sistema cumplió objetivos, beneficios, costos, adopción y requerimientos.

B

B2B

Comercio electrónico entre empresas, por ejemplo compra digital de insumos, servicios o componentes.

B2C

Comercio electrónico de empresa a consumidor, como una tienda online o una plataforma de suscripción.

Big Data

Manejo y análisis de grandes volúmenes, variedades y velocidades de datos para encontrar patrones y apoyar decisiones.

Bluetooth

Tecnología inalámbrica de corto alcance usada para conectar dispositivos personales, periféricos o sensores cercanos.

BPM

Sigla de administración de procesos de negocios. Enfoque para modelar, mejorar, ejecutar y medir procesos de manera continua.

Brecha digital

Desigualdad en acceso, habilidades o capacidad real de aprovechar tecnologías digitales.

Bugs Web

Elementos pequeños u ocultos en páginas o correos que pueden registrar visitas, aperturas o comportamiento.

C

C2C

Comercio electrónico entre consumidores, normalmente mediado por una plataforma con búsqueda, reputación y pago.

Calidad de datos

Grado en que los datos son correctos, completos, actuales, pertinentes y útiles para decidir.

Calidad de vida

Impacto de la tecnología en bienestar, acceso, autonomía, empleo, salud y relaciones sociales.

Calidad del sistema

Nivel de confiabilidad, seguridad, corrección y adecuación de un sistema para el uso esperado.

Cambio de paradigma

Transformación radical del modelo de operación o negocio habilitada por tecnología.

Cambio organizacional

Modificación en procesos, roles, reglas, responsabilidades, decisiones o cultura de trabajo dentro de una organización.

CASE

Herramientas de ingeniería de software asistida por computadora para documentar, modelar, generar, revisar y controlar artefactos de desarrollo.

Cliente/servidor

Modelo donde un cliente solicita datos o servicios y un servidor los entrega bajo reglas de comunicación y acceso.

Comercio digital

Intercambio de bienes, servicios, contenido o información mediado por canales digitales, plataformas, pagos, datos y modelos de ingresos.

Ciclo de vida tradicional

Método secuencial y formal de desarrollo de sistemas, con etapas definidas, documentación y aprobaciones antes de avanzar.

Comercio social

Uso de redes sociales, comunidades, reseñas y recomendaciones para atraer usuarios e influir en decisiones de compra.

Comisión por transacción

Modelo de ingresos donde una plataforma cobra una tarifa por cada venta, reserva, pedido, viaje u operación realizada.

Computación en la nube

Uso de aplicaciones, almacenamiento, plataformas o infraestructura entregados como servicio mediante redes.

Computación móvil

Acceso a datos, aplicaciones y procesos desde dispositivos portátiles conectados por redes inalámbricas o celulares.

Componentes

Piezas reutilizables de software que encapsulan funciones y pueden combinarse para construir aplicaciones.

Comunidad virtual

Espacio digital donde usuarios interactúan alrededor de intereses, identidades, productos o problemas comunes.

Conectividad

Capacidad de usuarios, dispositivos, aplicaciones y datos para comunicarse mediante redes.

Consentimiento informado

Aceptación basada en información clara sobre qué datos se usan, para qué, por quién y con qué consecuencias.

Continuidad operacional

Capacidad de mantener o recuperar procesos críticos cuando falla una red, servicio, proveedor, equipo o sistema.

Conmutación de paquetes

Forma de transmisión donde un mensaje se divide en paquetes que pueden viajar por distintas rutas y rearmarse al llegar.

Conversión

Paso del sistema antiguo al nuevo, incluyendo datos, procedimientos, capacitación, soporte y operación.

Conversión directa

Estrategia donde el sistema nuevo reemplaza al anterior en una fecha definida.

Conversión paralela

Estrategia donde el sistema antiguo y el nuevo operan temporalmente al mismo tiempo para comparar resultados y reducir riesgo.

Conversión piloto

Estrategia donde el sistema se implementa primero en un área limitada antes de extenderlo.

Conversión por fases

Estrategia donde el sistema se incorpora gradualmente por módulos, funciones, sedes o grupos de usuarios.

Cookies

Archivos o identificadores usados por sitios y servicios para recordar sesiones, preferencias o actividad.

Copyright

Protección legal relacionada con copia, distribución y uso autorizado de obras.

Costos de búsqueda

Tiempo, esfuerzo o dinero necesario para encontrar productos, precios, proveedores o información útil.

Costos de menú

Costos de cambiar y comunicar precios, catálogos o condiciones de venta.

Costos de transacción

Costos de negociar, coordinar, pagar, verificar, entregar o resolver una operación.

Creación de perfiles

Uso de datos para inferir intereses, hábitos, riesgos o características de una persona o grupo.

Creador de mercado

Actor digital que habilita un espacio donde terceros pueden publicar ofertas, buscar opciones y cerrar transacciones.

Crimen por computadora

Conducta ilegal realizada mediante sistemas, redes, dispositivos o datos.

D

Dato

Registro básico que puede transformarse en información cuando se organiza e interpreta en un contexto.

Densidad de información

Cantidad, calidad y disponibilidad de información útil para comparar, decidir y coordinar transacciones digitales.

Desintermediación

Reducción o eliminación de intermediarios tradicionales entre productor, vendedor y cliente.

Desarrollo ágil

Enfoque iterativo de desarrollo con entregas frecuentes, colaboración cercana con usuarios y adaptación continua a cambios.

Desarrollo de sistemas

Proceso organizado para analizar, diseñar, construir, probar, implementar y mantener una solución de información.

Desarrollo del usuario final

Creación de aplicaciones simples por usuarios no especialistas para resolver necesidades locales o de área.

Desarrollo móvil

Diseño y construcción de soluciones para dispositivos móviles, considerando pantallas pequeñas, sensores, permisos, conexión variable y uso en movimiento.

Desarrollo orientado a objetos

Enfoque que modela un sistema como objetos que combinan datos y comportamientos.

Derechos de autor

Protección de obras expresadas, como textos, software, música, imágenes o material audiovisual.

Derechos de información

Derechos y límites sobre recolección, acceso, uso, conservación y distribución de información.

Derechos de propiedad

Derechos sobre contenidos, software, datos, conocimiento o creaciones que pueden tener valor económico.

Dilemas éticos

Situaciones donde una decisión tecnológica produce beneficios y riesgos, sin una opción perfecta para todos.

Dirección IP

Identificador lógico que permite ubicar un dispositivo o servicio dentro de una red basada en IP.

Diseño de sistemas

Definición de cómo una solución cumplirá requerimientos mediante datos, procesos, interfaces, controles, reportes e infraestructura.

DNS

Sistema de nombres de dominio que traduce nombres legibles por personas a direcciones IP usadas por la red.

E

E-commerce

Compra, venta o intercambio de bienes, servicios, contenido o información mediante redes digitales, especialmente Internet y la Web.

E-tailer

Minorista en línea que vende directamente a consumidores desde su propio canal digital.

Ethernet

Familia de tecnologías usadas comúnmente en redes locales cableadas para transmitir datos entre dispositivos.

Estándares universales

Reglas técnicas compartidas que facilitan conectar compradores, vendedores, sistemas y servicios digitales.

Ética

Criterios para decidir qué acciones son correctas o incorrectas cuando afectan a otras personas.

Extranet

Extensión controlada de una red o sitio interno para dar acceso a actores externos autorizados.

F

Freemium

Modelo de ingresos donde una versión básica es gratuita y se cobra por funciones, contenido, capacidad o experiencia ampliada.

I

Igual a igual

Modelo de red donde los equipos comparten recursos directamente sin depender de un servidor central.

Información

Datos organizados que ayudan a comprender una situación o tomar decisiones.

Interactividad

Capacidad de comunicación bidireccional entre usuario, empresa, plataforma y otros usuarios.

Internet

Red global de redes que conecta sistemas y servicios mediante estándares comunes como TCP/IP.

Internet de las cosas

Conexión de objetos físicos con sensores, software y redes para registrar, enviar o usar datos del entorno.

IoT

Sigla de Internet de las cosas: objetos físicos conectados con sensores, software y redes para producir o usar datos.

Intranet

Red o sitio privado de una organización para compartir información, aplicaciones y servicios internos.

ISP

Sigla de proveedor de servicios de Internet. Organización que entrega acceso a Internet a hogares, personas o empresas.

L

LAN

Sigla de red de área local. Conecta dispositivos en un espacio cercano, como sala, oficina, edificio o campus.

Latencia

Demora entre una solicitud y una respuesta. Afecta videollamadas, juegos, sistemas en línea y aplicaciones interactivas.

M

Mantenimiento

Corrección, adaptación y mejora de un sistema después de su puesta en producción.

Marketplace

Plataforma que conecta compradores y vendedores externos, normalmente con mecanismos de búsqueda, pago, reputación y comisión.

Marketing conductual

Publicidad o personalización basada en comportamiento observado, como búsquedas, clics, compras, navegación o uso de una app.

Marketing dirigido

Publicidad, recomendación o mensaje adaptado según datos, segmentos e inferencias sobre un usuario.

Mercado digital

Espacio en línea donde compradores y vendedores pueden encontrarse, comparar información y realizar intercambios.

Metodologías estructuradas

Técnicas de análisis y diseño que modelan procesos, flujos de datos, módulos y transformaciones de forma ordenada.

M-commerce

Comercio electrónico realizado desde dispositivos móviles, con uso frecuente de apps, pagos móviles, notificaciones y ubicación.

Modelo de ingresos

Forma específica en que una organización obtiene dinero, como publicidad, suscripción, comisión, venta directa o afiliación.

Modelo de negocio

Forma en que una organización crea, entrega y captura valor.

N

Neutralidad de la red

Principio según el cual el tráfico de Internet debería tratarse sin discriminación injustificada por contenido, origen, destino o servicio.

NORA

Análisis que permite descubrir relaciones no evidentes entre datos, personas, eventos u organizaciones.

O

Outsourcing

Contratación externa de desarrollo, operación, soporte o servicios tecnológicos. Requiere control de requerimientos, costos, datos y desempeño.

P

Paquetes de software

Aplicaciones ya construidas que se adquieren, configuran y adaptan para resolver funciones comunes.

Patentes

Protección temporal sobre una invención que cumple requisitos legales.

Personalización

Adaptación de contenido, ofertas, recomendaciones o experiencia según datos del usuario.

Plataforma

Entorno digital que conecta participantes, servicios o transacciones bajo reglas comunes.

Plan de prueba

Documento o pauta que define condiciones, datos, resultados esperados, responsables y criterios para validar un sistema.

Portal

Punto de entrada digital que reúne búsqueda, contenido, servicios o enlaces para atraer tráfico y organizar acceso.

Privacidad

Control razonable sobre qué datos personales se recopilan, para qué se usan y con quién se comparten.

Producción

Operación real de un sistema con usuarios, datos, procesos y responsabilidades cotidianas.

Programación

Traducción del diseño de un sistema en código, configuración, automatización o integración técnica.

Producto digital

Bien que puede almacenarse, copiarse, distribuirse o consumirse en formato digital, como música, video, software, libros electrónicos o cursos.

Propiedad intelectual

Protección de creaciones, contenidos, software, marcas, métodos o conocimiento con valor cultural, técnico o económico.

Protocolo

Regla o estándar compartido que define cómo los sistemas se comunican, envían, reciben o interpretan datos.

Prototipado

Construcción rápida de una versión preliminar para aprender con usuarios, aclarar requerimientos y mejorar la solución.

Proveedor de contenido

Organización que produce, agrupa o distribuye contenido digital, como música, video, noticias o cursos.

Proveedor de servicios

Organización que entrega una función digital en línea, como transporte, entretenimiento, almacenamiento, pagos o gestión.

Proveedor de servicios de Internet

Empresa u organización que ofrece conexión a Internet y servicios relacionados. También se conoce como ISP.

Publicidad digital

Comunicación comercial o modelo de ingresos basado en espacios, audiencias, segmentación y medición en canales digitales.

Pruebas

Actividades para verificar que componentes, sistema completo y aceptación de usuarios cumplen resultados esperados.

R

RAD

Desarrollo rápido de aplicaciones mediante ciclos cortos, herramientas visuales, prototipos, reutilización y participación de usuarios.

Racionalización

Simplificación y mejora de procedimientos antes o junto con su automatización.

Red

Conjunto de dispositivos, medios, reglas y servicios que permite comunicar datos entre puntos.

Red celular

Infraestructura móvil que conecta teléfonos y dispositivos mediante antenas, celdas y operadores.

Red de área amplia

Red que conecta ubicaciones distantes, como sucursales, ciudades o redes de gran escala. También se conoce como WAN.

Red de área local

Red que conecta dispositivos dentro de un espacio cercano, como una sala, oficina, edificio o campus. También se conoce como LAN.

Red de telecomunicaciones

Conjunto de dispositivos, medios, reglas y servicios que permite transmitir datos, voz, video o señales entre puntos.

Red privada virtual

Conexión privada y cifrada creada sobre una red pública como Internet.

Redes de sensores inalámbricas

Conjunto de nodos con sensores que capturan datos del entorno y los envían por red para análisis o control.

Rendición de cuentas

Capacidad de identificar quién decidió, quién autorizó y quién debe explicar o corregir una consecuencia.

Rediseño de procesos

Replanteamiento profundo de un proceso para cambiar flujos de trabajo, pasos, responsabilidades y resultados.

Requerimiento

Necesidad o condición que orienta el diseño de una solución.

Requerimientos

Funciones, datos, reglas, restricciones y condiciones que una solución debe cumplir para resolver una necesidad.

Responsabilidad

Deber de hacerse cargo de una decisión, acción o consecuencia.

Responsabilidad legal

Obligación reconocida por leyes, contratos o tribunales cuando una acción o sistema causa daño.

RFID

Identificación por radiofrecuencia: tecnología para reconocer objetos, activos o personas mediante etiquetas y lectores.

Riqueza

Capacidad de entregar información con variedad de formatos, contexto y detalle, como texto, audio, video e interacción.

S

Secretos comerciales

Conocimiento valioso que una organización protege porque no es público y le da ventaja.

Seguimiento digital

Registro de actividades de una persona en sitios, aplicaciones, dispositivos o servicios.

Servicios basados en ubicación

Servicios móviles que usan posición aproximada o geográfica para ofrecer información, entrega, rutas, ofertas o coordinación.

Servicios Web

Servicios reutilizables accesibles por estándares de red para integrar aplicaciones y procesos.

Seguridad de red

Controles técnicos y organizacionales para proteger comunicaciones, dispositivos y servicios conectados.

Sistema de información

Conjunto de personas, procesos, datos y tecnología orientado a producir información útil.

Suscripción

Modelo de ingresos donde se cobra periódicamente por acceso continuo a contenido, servicio o funcionalidades.

T

TCP/IP

Familia de protocolos que permite conectar redes distintas y sostener el intercambio de datos en Internet.

Tecnoestrés

Estrés asociado al uso intensivo, permanente o invasivo de tecnologías digitales.

U

Ubicuidad

Disponibilidad de servicios digitales en distintos lugares y momentos, siempre que exista acceso a red y dispositivo.

UML

Lenguaje de modelado visual para representar requisitos, clases, interacciones, procesos y estructura de software.

V

VoIP

Voz sobre IP: transmisión de voz como datos digitales mediante redes basadas en protocolos de Internet.

VPN

Red privada virtual que crea un canal cifrado sobre una red pública para acceder de forma segura a recursos privados.

W

WAN

Sigla de red de área amplia. Conecta ubicaciones distantes, como sucursales, ciudades o redes de gran escala.

Web

Servicio de Internet basado en páginas, enlaces, navegadores y protocolos como HTTP.

Wi-Fi

Tecnología de red inalámbrica local basada en estándares 802.11, común para conectar dispositivos a una LAN o a Internet.

World Wide Web

Servicio de Internet basado en páginas, enlaces, navegadores y protocolos como HTTP.
